# 常用正则表达式
### 字符串提取数组
```
let str = "[微笑][哭泣]啊啊啊啊";
let result = str.match(/\[[^\]]*\]|[^[\]]+/g);
console.log(result); //['[微笑]', '[哭泣]', '啊啊啊啊']
```

### 是否是手机号
```
isphonenumber(e) {
    let reg = /^(13[0-9]|14[01456879]|15[0-35-9]|16[2567]|17[0-8]|18[0-9]|19[0-35-9])\d{8}$/
    return reg.test(e)
},
```
